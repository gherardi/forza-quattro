## Tech Stack

**Client:** HTML, Javascript,  TailwindCSS, Vite

**Server:** Node, Socket.IO, nodemon (only development)


## Run Locally

Clone the project

```bash
  git clone https://github.com/gherardi/socket
```

Go to the project directory

```bash
  cd socket
```

Install client and server dependencies

```bash
  cd client
  npm install
```

```bash
  cd server
  npm install
```

Start the server

```bash
  cd server
  npm run start
```

Start the client

```bash
  cd client
  npm run start
```
